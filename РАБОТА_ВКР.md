# Комплексная выпускная квалификационная работа
## Веб-приложение для проведения чемпионата по программированию IT-ВыСотка

---

## Общее описание проекта

Веб-приложение для проведения чемпионата по программированию IT-ВыСотка представляет собой комплексную платформу, обеспечивающую полный цикл организации мероприятия: от регистрации участников до генерации сертификатов и публикации результатов.

### Технологический стек

**Backend:**
- Node.js 20+ с Express 5.1
- PostgreSQL 15 (через Sequelize ORM)
- JWT для аутентификации
- Nodemailer для отправки email
- PDF-lib для генерации сертификатов
- Multer для загрузки файлов
- ExcelJS для экспорта данных

**Frontend:**
- React 19.2
- MobX для управления состоянием
- React Router 7 для маршрутизации
- Axios для HTTP-запросов

**Инфраструктура:**
- Docker и Docker Compose для контейнеризации
- Nginx как reverse proxy

### 1. Система аутентификации и регистрации

  - Обновление ссылки на эссе

  - Генерация access и refresh токенов

  - Валидация токенов
  - Хранение refresh токенов в БД

- **Middleware аутентификации** (`auth-middleware.js`)
  - Проверка JWT токенов

## Заключение

Проект представляет собой полнофункциональное веб-приложение для организации и проведения чемпионата по программированию. Работа была разделена между двумя разработчиками по принципу функциональных зон ответственности:

- **Якшибаев Д.И.** отвечал за пользовательскую (публичную) часть приложения, включая регистрацию, авторизацию, личный кабинет, работу с командами и все публичные страницы.

- **Мустаев В.Ф.** отвечал за административную часть приложения, включая управление участниками, аудиториями, рассадкой, генерацию сертификатов, файловый менеджер, настройки системы и управление результатами.

Оба разработчика совместно работали над общей инфраструктурой проекта (база данных, Docker, документация).

Проект успешно реализован с использованием современных технологий и лучших практик веб-разработки, обеспечивая удобный интерфейс как для участников, так и для администраторов чемпионата.
