# ОТЗЫВ РУКОВОДИТЕЛЯ

на выпускную квалификационную работу (бакалаврскую работу)

**Студент:** Якшибаев Данир Ильдарович  
**Группа:** БПО-22-02  
**Кафедра:** Вычислительной техники и инженерной кибернетики (ВТИК)  
**Направление подготовки:** 09.03.01 «Информатика и вычислительная техника»  
**Университет:** Уфимский государственный нефтяной технический университет (УГНТУ)

**Тема выпускной квалификационной работы:**  
«Разработка веб-приложения чемпионата по программированию IT-ВыСотка. Модуль пользователя»

---

Тема выпускной квалификационной работы Якшибаева Данира Ильдаровича является актуальной и практически значимой. Разработка модуля пользователя веб-приложения для чемпионата по программированию IT-ВыСотка направлена на автоматизацию процессов регистрации участников, управления командами и информирования о мероприятии. Ежегодно в чемпионате принимают участие более 200 школьников. Разработка предназначена для участников чемпионата (школьники 8-11 классов) и волонтеров. Веб-приложение применяется на кафедре ВТИК УГНТУ для обеспечения регистрации участников и управления командами.

Студентом спроектирован и реализован модуль пользователя веб-приложения. Студентом разработан механизм регистрации и авторизации участников с активацией аккаунтов по электронной почте, восстановлением паролей и использованием токенов доступа. Якшибаев Д.И. разработал личный кабинет участника с возможностью редактирования профиля и изменения формата участия. Создан механизм управления командами с генерацией уникальных пригласительных ссылок, что позволяет участникам самостоятельно формировать команды. Якшибаев Д.И. интегрировал в веб-приложение базу данных школ России, содержащую более 40000 записей, и реализовал каскадный выбор образовательного учреждения. Студент разработал публичные страницы с адаптивным дизайном для мобильных устройств. Особо следует отметить создание панели волонтера для контроля присутствия участников с возможностью массовой отметки по аудиториям и редактирования данных. Реализован механизм отправки электронных уведомлений участникам.

Выпускная квалификационная работа выполнена на высоком уровне и демонстрирует глубокое понимание студентом современных подходов к разработке веб-приложений. Разработанный модуль пользователя включает восемь функциональных блоков: регистрацию и авторизацию с активацией по электронной почте, восстановление пароля, личный кабинет участника, управление командами с пригласительными ссылками, интеграцию с базой школ России, публичные страницы, электронные уведомления и панель волонтера для контроля присутствия.

Техническая реализация отличается продуманной архитектурой. Студент применил компонентный подход при создании интерфейса. Интерфейс является современным и адаптирован для работы на устройствах с различными размерами экранов. Якшибаев Д.И. уделил значительное внимание вопросам информационной безопасности: реализована аутентификация на основе токенов доступа, криптографическое хеширование паролей, защита маршрутов, валидация входных данных, защита от внедрения вредоносного кода. Веб-приложение развернуто с использованием технологии контейнеризации.

Пояснительная записка оформлена в соответствии с требованиями ГОСТ. Текст написан грамотным техническим языком. Работа содержит необходимые иллюстративные материалы. Список использованных источников содержит актуальную литературу по веб-разработке. Существенных замечаний к выполненной работе нет.

Выпускная квалификационная работа Якшибаева Данира Ильдаровича соответствует требованиям, предъявляемым к работам данного уровня. Студент продемонстрировал высокий уровень теоретических знаний и практических навыков в области разработки современных веб-приложений. Разработанный модуль обладает практической значимостью и успешно применяется для автоматизации процессов регистрации участников и проведения чемпионата по программированию IT-ВыСотка. Рекомендуемая оценка — «отлично». Якшибаев Данир Ильдарович рекомендуется к присвоению квалификации «бакалавр» по направлению подготовки 09.03.01 «Информатика и вычислительная техника».
