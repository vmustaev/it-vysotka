# ОТЗЫВ РУКОВОДИТЕЛЯ

на выпускную квалификационную работу (бакалаврскую работу)

**Студент:** Якшибаев Данир Ильдарович  
**Группа:** БПО-22-02  
**Кафедра:** Вычислительной техники и инженерной кибернетики (ВТИК)  
**Направление подготовки:** 09.03.01 «Информатика и вычислительная техника»  
**Университет:** Уфимский государственный нефтяной технический университет (УГНТУ)

**Тема выпускной квалификационной работы:**  
«Разработка веб-приложения чемпионата по программированию IT-ВыСотка. Модуль пользователя»

---

Выпускная квалификационная работа Якшибаева Данира Ильдаровича посвящена автоматизации процессов регистрации участников и управления командами чемпионата по программированию IT-ВыСотка, проводимого кафедрой ВТИК УГНТУ для более 200 школьников 8-11 классов. Разработка предназначена для участников чемпионата и волонтеров. Веб-приложение применяется на кафедре ВТИК УГНТУ.

Студентом спроектирован и реализован модуль пользователя веб-приложения: механизм регистрации и авторизации с активацией по электронной почте, восстановлением паролей и токенами доступа, личный кабинет участника с редактированием профиля, управление командами с генерацией пригласительных ссылок. Интегрирована база данных школ России (более 40000 записей) с каскадным выбором учреждения. Разработаны публичные страницы с адаптивным дизайном, панель волонтера для контроля присутствия с массовой отметкой по аудиториям, механизм отправки электронных уведомлений.

Модуль пользователя включает восемь функциональных блоков: регистрацию и авторизацию с активацией по электронной почте, восстановление пароля, личный кабинет участника, управление командами с пригласительными ссылками, интеграцию с базой школ России, публичные страницы, электронные уведомления и панель волонтера.

Интерфейс создан с использованием компонентного подхода и адаптирован для устройств с различными размерами экранов. Реализована аутентификация на основе токенов доступа, криптографическое хеширование паролей, защита маршрутов, валидация входных данных, защита от внедрения вредоносного кода. Веб-приложение развернуто с использованием контейнеризации.

Пояснительная записка оформлена в соответствии с требованиями ГОСТ, содержит необходимые иллюстративные материалы и актуальную литературу по веб-разработке. Существенных замечаний нет.

Выпускная квалификационная работа Якшибаева Данира Ильдаровича соответствует требованиям, предъявляемым к работам данного уровня. Разработанный модуль обладает практической значимостью и применяется для автоматизации процессов регистрации участников и проведения чемпионата IT-ВыСотка. Рекомендуемая оценка — «отлично». Якшибаев Данир Ильдарович рекомендуется к присвоению квалификации «бакалавр» по направлению подготовки 09.03.01 «Информатика и вычислительная техника».
