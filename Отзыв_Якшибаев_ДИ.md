# ОТЗЫВ НАУЧНОГО РУКОВОДИТЕЛЯ

на выпускную квалификационную работу (бакалаврскую работу)

**Студент:** Якшибаев Данир Ильдарович  
**Группа:** БПО-22-02  
**Кафедра:** Вычислительной техники и инженерной кибернетики (ВТИК)  
**Направление подготовки:** 09.03.01 «Информатика и вычислительная техника»  
**Университет:** Уфимский государственный нефтяной технический университет (УГНТУ)

**Тема выпускной квалификационной работы:**  
«Веб-приложение чемпионата по программированию IT-ВыСотка. Модуль пользователя»

---

Тема выпускной квалификационной работы Якшибаева Данира Ильдаровича является актуальной и практически значимой. Разработка модуля пользователя веб-приложения для чемпионата по программированию IT-ВыСотка направлена на решение задачи автоматизации процессов регистрации участников, управления командами и информирования о мероприятии. Ежегодно в чемпионате принимают участие более 200 школьников из различных образовательных учреждений. Данная разработка предназначена для участников чемпионата по программированию IT-ВыСотка (школьники 8-11 классов) и волонтеров для контроля присутствия участников в день проведения мероприятия. Веб-приложение применяется на кафедре ВТИК УГНТУ для обеспечения регистрации участников, управления командами и доступа к информации о чемпионате.

Якшибаев Д.И. самостоятельно спроектировал и реализовал модуль пользователя веб-приложения. Студентом разработан полнофункциональный механизм регистрации и авторизации участников с активацией аккаунтов по электронной почте, восстановлением забытых паролей и использованием токенов доступа для обеспечения безопасности. Якшибаев Д.И. разработал личный кабинет участника с возможностью редактирования профиля, изменения формата участия и добавления ссылки на эссе. Создан оригинальный механизм управления командами с генерацией уникальных пригласительных ссылок, что позволяет участникам самостоятельно формировать команды без участия администраторов. Якшибаев Д.И. интегрировал в веб-приложение базу данных школ России, содержащую более 40000 записей, и реализовал каскадный выбор образовательного учреждения с фильтрацией по регионам и городам. Студент разработал публичные страницы веб-приложения с адаптивным дизайном для мобильных устройств, планшетов и настольных компьютеров. Особо следует отметить создание панели волонтера для контроля присутствия участников в день проведения чемпионата с возможностью массовой отметки присутствия по аудиториям, редактирования данных участников и экспорта списков. Реализован механизм отправки электронных уведомлений участникам на различных этапах взаимодействия с веб-приложением.

Выпускная квалификационная работа выполнена на высоком уровне и демонстрирует глубокое понимание студентом современных подходов к разработке веб-приложений. Якшибаев Д.И. показал владение современными технологиями для создания клиентской части и реализации серверной логики. Разработанный модуль пользователя включает восемь основных функциональных блоков: механизм регистрации и авторизации с активацией по электронной почте, механизм восстановления пароля, личный кабинет участника, механизм управления командами с пригласительными ссылками, интеграцию с базой школ России с каскадным выбором, публичные страницы веб-приложения, механизм электронных уведомлений и панель волонтера для контроля присутствия участников.

Техническая реализация отличается продуманной архитектурой и соответствием современным стандартам веб-разработки. Студент применил компонентный подход при создании интерфейса, что обеспечивает переиспользование кода и упрощает поддержку веб-приложения. Интерфейс веб-приложения является современным, интуитивно понятным и адаптирован для работы на устройствах с различными размерами экранов. Якшибаев Д.И. уделил значительное внимание вопросам информационной безопасности и защиты персональных данных участников: реализована аутентификация на основе токенов доступа с ограниченным сроком действия, криптографическое хеширование паролей, защита маршрутов, валидация входных данных на стороне клиента и сервера, защита от внедрения вредоносного кода. Веб-приложение развернуто с использованием технологии контейнеризации.

Пояснительная записка к выпускной квалификационной работе оформлена в соответствии с требованиями ГОСТ и методическими указаниями УГНТУ. Текст написан грамотным техническим языком с использованием корректной терминологии. Работа содержит необходимые иллюстративные материалы: диаграммы взаимодействия компонентов, схемы базы данных, скриншоты пользовательского интерфейса. Список использованных источников содержит актуальную литературу по веб-разработке и официальную документацию используемых технологий. Существенных замечаний к выполненной работе нет.

Выпускная квалификационная работа Якшибаева Данира Ильдаровича соответствует всем требованиям, предъявляемым к работам данного уровня. Студент продемонстрировал высокий уровень теоретических знаний и практических навыков в области разработки современных веб-приложений. Разработанный модуль пользователя обладает практической значимостью и успешно применяется для автоматизации процессов регистрации участников и проведения чемпионата по программированию IT-ВыСотка.

Рекомендуемая оценка – «отлично».
