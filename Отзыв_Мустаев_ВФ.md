# ОТЗЫВ НАУЧНОГО РУКОВОДИТЕЛЯ

на выпускную квалификационную работу (бакалаврскую работу)

**Студент:** Мустаев Венер Фанисович  
**Группа:** БПО-22-02  
**Кафедра:** Вычислительной техники и инженерной кибернетики (ВТИК)  
**Направление подготовки:** 09.03.01 «Информатика и вычислительная техника»  
**Университет:** Уфимский государственный нефтяной технический университет (УГНТУ)

**Тема выпускной квалификационной работы:**  
«Веб-приложение чемпионата по программированию IT-ВыСотка. Модуль администратора»

---

Выпускная квалификационная работа Мустаева Венера Фанисовича посвящена актуальной задаче автоматизации организационных процессов проведения образовательных мероприятий в области информационных технологий. Чемпионат по программированию IT-ВыСотка проводится кафедрой ВТИК УГНТУ ежегодно и охватывает более 200 школьников 8-11 классов из различных регионов России. Данная разработка предназначена для организаторов чемпионата по программированию IT-ВыСотка, администраторов образовательных мероприятий и учебных заведений, проводящих олимпиады и конкурсы. Веб-приложение применяется на кафедре ВТИК УГНТУ для автоматизации всех этапов организации и проведения чемпионата.

Мустаев В.Ф. самостоятельно спроектировал и реализовал модуль администратора веб-приложения. Студентом разработана архитектура модуля, спроектирована структура базы данных для хранения информации об участниках, аудиториях, командах и результатах, реализована серверная и клиентская части приложения. Особо следует отметить разработку алгоритма автоматической рассадки участников по аудиториям с учетом командного формата участия, а также реализацию механизма генерации персонализированных сертификатов с интерактивной настройкой позиционирования текста. Мустаев В.Ф. реализовал модуль резервного копирования базы данных, файловый менеджер для управления контентом, механизм управления волонтерами и настройками веб-приложения. Обеспечена безопасность модуля администратора через механизм аутентификации и авторизации с разграничением доступа по ролям.

Выпускная квалификационная работа выполнена на высоком техническом уровне. Студент продемонстрировал глубокие знания современных веб-технологий, умение работать с серверными и клиентскими фреймворками, навыки проектирования баз данных. Разработанный модуль администратора включает десять основных функциональных блоков: главную панель управления с отображением статистики, модуль управления участниками с возможностью фильтрации и экспорта данных, модуль управления аудиториями, модуль автоматической и ручной рассадки участников, модуль генерации сертификатов с интерактивной настройкой шаблона, файловый менеджер, модуль управления настройками веб-приложения и регистрацией, модуль управления результатами чемпионата с автоматической отправкой уведомлений победителям, модуль резервного копирования базы данных и модуль управления волонтерами.

Техническое решение отличается продуманной архитектурой, соответствующей принципам разделения ответственности и модульности. Интерфейс модуля администратора является интуитивно понятным, содержит боковое меню навигации с группировкой функций по разделам, всплывающие уведомления об успешных операциях и ошибках, модальные окна для подтверждения критических операций. Студент уделил должное внимание вопросам информационной безопасности: реализована аутентификация с использованием токенов доступа, криптографическое хеширование паролей, разграничение доступа по ролям, защита от внедрения вредоносного кода, валидация входных данных. Разработанное веб-приложение развернуто с использованием технологии контейнеризации.

Пояснительная записка к выпускной квалификационной работе оформлена в соответствии с требованиями ГОСТ и методическими указаниями университета. Текст изложен грамотным техническим языком. Работа содержит необходимые графические материалы: диаграммы архитектуры веб-приложения, схемы базы данных, скриншоты интерфейса. Список использованных источников включает актуальную техническую литературу и официальную документацию используемых технологий. Существенных замечаний к выполненной работе нет.

Выпускная квалификационная работа Мустаева Венера Фанисовича соответствует всем требованиям, предъявляемым к работам данного уровня. Студент продемонстрировал высокий уровень теоретической подготовки и практических навыков в области разработки веб-приложений. Разработанный модуль администратора представляет практическую ценность и успешно применяется для автоматизации организационных процессов проведения чемпионата по программированию IT-ВыСотка.

Рекомендуемая оценка – «отлично».
